# Moe's Frontmatters

Quartz 4 portfolio site. Game writing, film production, marketing.

## Commands

```bash
npx quartz build          # Build to public/
npx quartz build --serve  # Dev server with hot reload
```

## Structure

- `content/` — Markdown source files
- `content/Digital Garden/` — Project case studies
- `quartz/` — Quartz framework (components, plugins)
- `quartz.config.ts` — Site config (baseUrl, theme, plugins)
- `private/` — Reference docs, not published

## Quartz Behavior

- **Slugs**: Derived from filename, preserving case
- **Permalinks**: Create alias redirects, not canonical URL changes
- **OG images**: Generated by `CustomOgImages` plugin

## Frontmatter

Required for published pages:
- `title` — Display title (nav, breadcrumbs, page header). 60 chars max
- `description` — 155 chars max
- `tags` — Array `[tag1, tag2]`
- `draft: true` — Excludes from build

Optional overrides (custom, not native Quartz):
- `seoTitle` — Overrides `title` in `<title>`, `og:title`, `twitter:title`. Nav elements still use `title`
- `socialDescription` — Overrides `description` in OG/Twitter tags (native Quartz)
